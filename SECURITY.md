# Security Policy

## Supported Versions

InterKnot-Web is under active development.

| Version                  | Supported |
| ------------------------ | --------- |
| Latest (`main`)          | ✅         |
| Older commits / releases | ❌         |

Only the latest version is maintained and receives security updates.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in InterKnot-Web, please report it responsibly.

* Include as much information as possible, such as:

  * A description of the issue
  * Steps to reproduce
  * Any proof-of-concept, logs, or screenshots

After receiving a report, the maintainer will:

1. Confirm receipt of the report.
2. Investigate and validate the issue.
3. Develop and release a fix if necessary.
